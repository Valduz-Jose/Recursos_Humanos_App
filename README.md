# 👥 Sistema de Gestión de Recursos Humanos (Full Stack)

Este es un proyecto Full Stack diseñado para la gestión de empleados, permitiendo realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) de forma eficiente.

## 🚀 Demo en Vivo
- **Frontend:** recursos-humanos-app-rust.vercel.app
- **Backend API:** https://rh-backend-python.onrender.com

## 🛠️ Tecnologías Utilizadas

### Frontend
- **Angular 17**: Framework principal.
- **TypeScript**: Programación robusta.
- **Bootstrap**: Diseño responsivo y moderno.

### Backend
- **Python 3.11**: Lenguaje de servidor.
- **Flask**: Micro-framework para la API REST.
- **SQLAlchemy**: ORM para la gestión de base de datos.
- **Marshmallow**: Serialización de datos.

### Base de Datos & Despliegue
- **PostgreSQL**: Base de datos relacional de producción.
- **Render**: Hosting del Backend y DB.
- **Vercel**: Hosting del Frontend.

## 📋 Características
- ✅ Visualización de lista de empleados en tiempo real.
- ✅ Registro de nuevos empleados con validaciones.
- ✅ Edición y eliminación de registros.
- ✅ Conexión segura a base de datos en la nube.
- ✅ Configuración de CORS para comunicación entre dominios.

## 🔧 Instalación Local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Valduz-Jose/Recursos_Humanos_App.git
   ```

2.Configura el backend:

```bash
cd backend
pip install -r requirements.txt
python app.py
```

3.Configura el frontend:

```bash
cd frontend
npm install
ng serve
```
